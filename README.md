# 🚀 HabitForge

HabitForge is a full-stack MERN application that combines habit tracking with AI-generated coding challenges and quizzes. Instead of being just another habit tracker, the goal was to make learning and staying consistent more engaging through gamification, collaboration, and real-time competition.

The idea behind HabitForge was simple: learning is easier when you're consistent, and consistency is easier when you're motivated. By combining daily challenges, coding practice, quizzes, streaks, and leaderboards into one platform, HabitForge encourages users to build productive habits while improving their technical skills.

---

# ✨ Features

### 🤖 AI-Generated Challenges

One of the core features of HabitForge is AI-powered challenge generation.

Users can create:

* Coding challenges
* Multiple-choice quizzes

Challenges are generated based on the selected topic and difficulty level, making every learning session unique.

---

### 💻 Coding Practice

HabitForge provides a built-in coding environment where users can solve problems without leaving the platform.

Features include:

* Monaco Editor integration
* Live code execution using JDoodle API
* Automatic test case validation
* Support for multiple programming languages

---

### 📝 Interactive Quizzes

The platform also supports AI-generated quizzes with instant evaluation.

Users receive:

* Multiple-choice questions
* Immediate scoring
* Performance tracking
* Fresh questions through regeneration

---

### 👥 Group Challenges

Users can create groups and invite friends using a unique group code.

Each group can:

* Participate in shared challenges
* Track member progress
* Compete on leaderboards
* Build streaks together

This makes learning more interactive and competitive.

---

### 🔥 Streak & Points System

To encourage consistency, HabitForge includes:

* Daily streak tracking
* Smart point system
* Group streaks
* Leaderboards

Challenges can also be regenerated by the group creator, allowing members to continue practicing with new content while keeping the overall group history intact.

---

### 📊 Analytics Dashboard

The dashboard gives users a quick overview of their progress through charts and statistics, making it easy to track consistency and improvement over time.

---

### 🔐 Authentication

The application includes secure authentication using JWT along with protected routes and encrypted passwords using bcrypt.

# 💡 Challenges I Solved

This project taught me much more than just building a MERN application.

Some of the most interesting problems I worked on were:

* Designing MongoDB schemas for groups, members, habits, and completions
* Managing multiple users attempting the same challenge
* Building a streak system that updates correctly for individuals and groups
* Implementing daily challenge regeneration while resetting only the current challenge cycle instead of the overall group progress
* Integrating AI-generated coding challenges and quizzes into the application
* Validating code execution against dynamic test cases
* Keeping leaderboards and scores synchronized across different users

# 🚀 Running the Project

### Clone the repository

```bash
git clone https://github.com/Ayush310306/HabitForge.git
```

### Backend

```bash
cd backend
npm install
npm run server
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

# Environment Variables

Create a `.env` file inside the backend folder and add:

```env
MONGODB_URI=
JWT_SECRET=
EMAIL_USER=
EMAIL_PASS=
OPENROUTER_API_KEY=
JDOODLE_CLIENT_ID=
JDOODLE_CLIENT_SECRET=
```
